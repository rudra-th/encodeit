# encodeit
A message encoder and decoder which works on a key which can be shared between people who want to decode or encode messages which can be transfered without fear of anyone reading, also the encoding takes place locally on your device meaning there is no server relation which ensure your messages are safe
🔐 encodeit | The ultimate minimalist web utility for instant data encoding and decoding. Fast, secure, and entirely client-side. No tracking, no servers, just pure utility.
